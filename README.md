## 방탈출 사용자 예약

### 요구사항

- **4 단계**
    - 사용자 도메인을 추가합니다.
    - 로그인 기능을 구현하세요.
    - 로그인 후 Cookie를 이용하여 사용자의 정보를 조회하는 API를 구현하세요.

### 세부 요구 사항

#### 로그인 기능

- [ ] 사용자가 이메일과 비밀번호를 입력하여 로그인 할 수 있다.
    - [ ] 로그인 시 쿠키를 함께 보낸다.
- [ ] 토큰 값으로 사용자의 이름을 가져올 수 있다.
